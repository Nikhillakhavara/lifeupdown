📌 Step-by-Step Guide

1. Export Instagram Data

Go to Instagram → Settings → Privacy & Security → Download Data. Request HTML format. Extract the ZIP and find messages.html.

2. Open Converter

Double-click this converter.html file in your browser to launch the tool.

3. Upload File

Click Choose File and select messages.html from Instagram data.

4. Convert & Download

Click Convert & Download Excel. Your chats will download as instagram_chat.xlsx.
